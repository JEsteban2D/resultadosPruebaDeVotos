(Tecnologias):
Se realizo la prueba usando la tecnologia React en el front junto con javascript HTML y CSS,
asi como Nodejs en el back y MYSQL para la creacion de la nase de datos, junto con librerias 
como axios para los metodos http y comunicarnos con la base de datos desde React,
uso de la libreria Sequelize que permite la manipulacion de bases de datos,
y express en conjuntos con Nodejs.

(Start):
para comenzar la aplicacion se debera montar la base de datos anexada en el documento DB
e iniciar el servidor (xampp en caso de local)
al estar dentro de la carpeta app,en dos terminales diferentes se deberan correr los siguientes comandos:
*cd node (+la tecla ENTER seguido de:) nodemon app
*cd reactappfront (+la tecla ENTER seguido de:) npm start

(Consideraciones):
el proyecto mostrara el logo de React juntos con los datos organizados en una tabla que son traidos
desde la base de datos mostrara la hora actual en la que se encuentra a la hora de ejecutar y un boton
que mostrara el arreglo de la base de datos en consola.
**La idea es mostrar el resultado de las votaciones al dar click en el boton y desabilitar este en las horas especificadas pero por falta de experiencia
no pude dar resolucion a este  problema en el tiempo que se dispuso para la prueba.
